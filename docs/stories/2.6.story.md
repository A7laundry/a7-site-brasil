## Estrutura de cada LP

Cada LP deve ter:
1. **Hero** — título H1 com keyword, subtítulo, CTA WhatsApp primário
2. **Como funciona** — 3 passos: agendar → coleta → entrega
3. **Diferencial** — específico para o item (ex: tênis: sem deformação; terno: sem brilho)
4. **Preço / prazo** — tabela simples ou range
5. **FAQ** — 3-4 perguntas frequentes com schema FAQ (JSON-LD)
6. **CTA final** — WhatsApp + endereço
